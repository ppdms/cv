# Basil Papadimas — CV

LaTeX resume based on [Jake's Resume template](https://github.com/jakegut/resume).

![CV render](CV_Basil_Papadimas.png)

## Building

1. Install Nix
2. Run `make build`

The Nix flake provides all required TeX Live packages.

To regenerate the README preview image, run `make render`.
